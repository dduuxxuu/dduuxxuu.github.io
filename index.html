利用session制作验证码

Posted on 2018-01-14 | Edited on 2019-03-23 | In PHP

利用php中的session制作验证码
验证码的运用在登录界面运用很频繁。利用session实现了验证码的验证。

  • 利用php生成一张图片
  • 随机生成验证码字符串
  • 将字符串存在session里
  • 将字符串随机添加颜色大小角度加载到图片里
  • 为验证码添加干扰元素
  • 输出图片
  • 验证端输入字符，对其与session里保存字符进行比较，输出结果

//php内容

<?php 
/*实现验证码功能*/ 
//开启session 
//启用session
session_start(); 
//创建一个大小为 100*30 的验证码图片 
$image = imagecreatetruecolor(100, 30); 
$bgcolor = imagecolorallocate($image, 255, 214, 118);
//将图片区域填充 
imagefill($image, 0, 0, $bgcolor); 
$captch_code = ''; 
for ($i = 0; $i < 4; $i++) { 
    //定义字体大小
    $fontSize = 8; 
    //随机定义字体颜色
    $fontColor = imagecolorallocate($image, rand(0, 255), rand(0, 255), rand(0, 255)); 
    //定义验证码取值，排除易混字
    $data = '3456789abcdefghijkmnpqrstuvwxy'; 
    //随机取字
    $fontContainer = substr($data, rand(0, strlen($data) - 1), 1);
    //拼接字
    $captch_code .= $fontContainer; 
    $x = ($i * 100 / 4) + rand(5, 10); 
    $y = rand(5, 10); 
    imagestring($image, $fontSize, $x, $y, $fontContainer, $fontColor); 
} 
//就生成的验证码保存到session 
$_SESSION['randcode'] = $captch_code; 
//在图片上增加点干扰元素 
for ($i = 0; $i < 300; $i++) { 
    $pointColor = imagecolorallocate($image, rand(50, 200), rand(50, 200), rand(50, 200)); imagesetpixel($image, rand(1, 99), rand(1, 29), $pointColor); 
} 
//在图片上增加线干扰元素 
for ($i = 0; $i < 8; $i++) { 
    $lineColor = imagecolorallocate($image, rand(80, 220), rand(80, 220), rand(80, 220)); imageline($image, rand(1, 99), rand(1, 29), rand(1, 99), rand(1, 29), $lineColor); 
} 
//设置头 
header('content-type:image/png'); 
imagepng($image); 
imagedestroy($image); 

?>

jQuery插件的简单封装

Posted on 2018-01-10 | Edited on 2019-03-23 | In jQuery

jQuery插件的封装总结起来就是三步

  • 使用html, css, js, jquery成功实现部件的完整效果
  • 将css拷贝到css文件中
  • 为jQuery.fn添加实例方法

以下是一个下拉菜单的封装

//html
<div id="my-dropdown" class="dropdown">
<a data-toggle="dropdown" href="#">Dropdown</a>
<ul class="dropdown-menu fade">
    <li><a href="#">Action</a></li>
    <li><a href="#">Another action</a></li>
    <li><a href="#">Something else here</a></li>
    <li><hr></li>
    <li><a href="#">Separated link</a></li>
</ul>
</div>



//css
.dropdown{width:64px; height:20px;}
.dropdown a{
    text-decoration:none; 
    color:#000;
    display:inline-block;
    }
.dropdown>:first-child{ 
    border:1px solid #000; padding:6px 12px;
    }
.dropdown>.dropdown-menu{
    width:200px;
    list-style:none; margin:0; padding:0;
    border:1px solid #aaa;
    box-shadow:5px 5px 6px #000;
    }
.dropdown>.dropdown-menu a{padding:6px 20px; width:160px}
.dropdown-menu li:hover{background:#ddd}
.fade{
    height:0;
    opacity:0;
    overflow:hidden;
    transition:all .5s linear;
}
.in{
    height:152px;
    opacity:1;
}

//js

if(typeof jQuery!=="function")
    throw new Error("du-dropdown依赖于jquery")
else{
    jQuery.fn.dropdown = function(){
        //侵入：为制定对象添加class和data-*
    var $parent=this;
    $parent.addClass("dropdown")
        .children()
        .next().addClass("dropdown-menu fade");
    //添加事件
    $parent.hover(function(){
        var $parent = $(this);
        $parent.children(".dropdown-menu").toggleClass("in");
    })
     }
}

AJAX封装

Posted on 2017-12-28 | Edited on 2019-03-23 | In JavaScript

AJAX函数封装

由于ajax的调用在网站创建过程中使用极为频繁,故进行封装。封装中主要解决以下问题。

  • 判定浏览器环境，创建XMLHttpRequest
  • 对使用get或者post方法进行判定，针对不同方法处理url
  • 设置回调函数，如果返回为json文件进行处理
  • 使用回调函数处理返回结果
  • 针对不同的方法修改请求头，发送主体
  • 使用参数解构的方法，将参数放入对象，调用更灵活
  • 为防止参数中的回调函数发生回调地狱，利用Promise技术进行优化

代码如下

function ajax({type,url,data,dataType}){
    return new Promise(callback=>{
    //1.创建xhr对象
    if(window.XMLHttpRequest){
        var xhr=new XMLHttpRequest(); //标准创建
    }else{
        //IE8以下的创建方式
        var xhr=new ActiveXObject("Microsoft.XMLHttp");
    }
    //如果type是get需要处理url
    if(type.toLowerCase()=="get"&&data!==undefined) 
        url+="?" + data;
    //2.创建请求
    xhr.open(type,url,true);
    //3.设置回调函数
    xhr.onreadystatechange=function(){
        //判断readyState以及status
        if(xhr.readyState==4){
          if(xhr.status==200){
            //接收响应数据
            var resData=xhr.responseText;
             //如果是json格式,进行处理
            if(dataType!==undefined&&dataType.toLowerCase()=="json") 
                resData=JSON.parse(resData);
            callback(resData);
          }
        }   
    }
    //如果是post增加请求
    if(type.toLowerCase()==="post")  
        xhr.setRequestHeader("Content-Type","application/x-www-form-urlencoded");
    //4.发送xhr
    xhr.send(type.toLowerCase()=="get"?null:data);
 });
}

调用时需要注意回调的使用，举例

ajax({
    type:"get",
    url:"server.php",
    data:"uname=dduuxxuu&upwd=123456",
    dataType:"json",     
}).then(resData=>{
        console.log(resData);
})
